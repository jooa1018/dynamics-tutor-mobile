# 현재 한계 및 사용 범위

DynaTutor 개인용 모바일 웹앱은 **동역학 문제의 유형 판별, 적용식/비적용식 안내, FBD 점검, 풀이 순서 안내, 오답노트/복습**을 돕는 학습 보조 도구입니다. 모든 교재 문제를 자동으로 읽고 최종 수치 정답까지 계산하는 완전 자동 풀이기는 아닙니다.

## 핵심 한계

1. **이미지/그림 직접 인식 불가**  
   교재 그림에 포함된 치수, 각도, 힘 방향, 접촉 조건, 도르래 연결 구조를 텍스트로 입력해야 합니다.

2. **완전 자동 수치 풀이 제한**  
   대표 계산 모듈은 있으나 모든 미지수 추출, 방정식 자동 구성, 단위 변환, 연립방정식 풀이, 유효숫자 포함 최종 답 산출은 제한적입니다.

3. **3D 강체 운동 미지원**  
   자이로스코프, 오일러 방정식, 관성텐서, 세차/장동, 회전좌표계, 3D 코리올리 문제는 현재 미지원입니다.

4. **지원 유형 제한**  
   2D 입자 동역학, 강체 평면운동, 원운동, 구름운동, 경사진 커브, 원뿔진자, 블록-도르래, 회전충돌 대표 유형 중심입니다.

5. **Streamlit 모바일 웹앱 한계**  
   모바일 브라우저에서 사용할 수 있지만 네이티브 앱/PWA가 아니며, 오프라인 사용·푸시 알림·앱스토어 설치·네이티브 수준 터치 인터랙션은 지원하지 않습니다.

## 좋은 입력 원칙

- 그림 속 정보를 텍스트로 풀어 씁니다.
- 물체, 질량, 각도, 반지름, 마찰, 힘 방향, 지지 방식, 구하려는 값을 명시합니다.
- theta, omega, mu_s 같은 ASCII 표현을 사용해도 됩니다.
