# 스마트폰 홈 화면 바로가기 안내

앱스토어 앱을 만들지 않고도 스마트폰 홈 화면에서 앱처럼 사용할 수 있습니다.

## iPhone Safari

1. Safari로 배포된 앱 URL에 접속합니다.
2. 하단 공유 버튼을 누릅니다.
3. **홈 화면에 추가**를 선택합니다.
4. 이름을 `DynaTutor`로 지정하고 추가합니다.

## Android Chrome

1. Chrome으로 배포된 앱 URL에 접속합니다.
2. 우측 상단 메뉴 버튼을 누릅니다.
3. **홈 화면에 추가**를 선택합니다.
4. 이름을 `DynaTutor`로 지정하고 추가합니다.

## 사용 팁

- 첫 접속 시 앱 비밀번호를 입력합니다.
- 모바일에서는 전문가/디버그 정보가 기본적으로 접힘 상태입니다.
- 오답노트는 주기적으로 CSV/JSON으로 백업하세요.
