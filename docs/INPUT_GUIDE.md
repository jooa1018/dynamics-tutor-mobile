# 모바일 문제 입력 가이드

## 기호 입력

특수기호를 직접 입력하지 않아도 ASCII 표현을 사용할 수 있습니다.

| 입력 | 의미 |
|---|---|
| theta | θ |
| omega | ω |
| alpha | α |
| mu_s | μ_s |
| mu_k | μ_k |
| sum F | ΣF |
| sum M_G | ΣM_G |
| e_theta | e_θ |
| theta_dot | θ_dot |

## 좋은 입력 예시

> A 2 kg block moves on a frictionless incline of angle theta = 30 deg. Find acceleration along the incline.

> A bob tied to a string of length L moves in a horizontal circle. The string makes angle theta with the vertical. Find omega.

## 입력 템플릿

앱의 문제 입력 화면에서 수평면 블록-도르래, 경사면, 수직 원운동, 원뿔진자, 경사진 커브, 순수 구름, 회전충돌 템플릿 버튼을 제공합니다.
