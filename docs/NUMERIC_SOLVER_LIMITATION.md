# 계산 지원 범위

이 앱은 동역학 문제의 유형을 판별하고, 적용식과 풀이 순서를 안내하는 학습 보조 도구입니다. 모든 문제의 최종 수치 답을 자동으로 계산하는 완전 자동 풀이기는 아닙니다.

## 지원 수준

- 유형 판별: 가능
- 적용식 제시: 가능
- 풀이 골격 제시: 가능
- 자동 수치 답 계산: 제한적
- 단위/유효숫자 포함 최종 답: 제한적

## 현재 계산 도우미

- 등가속도 직선운동
- 포물선 운동 기본식
- 높이-속도 에너지
- 수직 원운동 최고점 최소속도
- 1차원 충돌
- 순수 구름 에너지

복잡한 연립방정식, 단위 변환, 모든 미지수 자동 추출은 사용자가 직접 확인해야 합니다.
