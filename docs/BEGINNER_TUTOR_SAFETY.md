# 입문자 안전 튜터 모드

이 앱은 자동 정답 풀이기가 아니라, 동역학 문제를 읽고 어떤 원리와 FBD를 먼저 세워야 하는지 안내하는 튜터형 도구입니다.

## 정보 부족 입력 처리

다음처럼 그림 정보가 빠진 입력은 풀이 추천을 중단합니다.

> 그림과 같은 시스템에서 가속도를 구하라.

출력 정책:

- 정보 부족 상태를 명확히 표시
- 특정 공식/풀이법을 확정 추천하지 않음
- 물체 개수, 질량, 마찰, 줄/도르래, 운동 방향, 구하려는 값 체크리스트 제공
- 조건 보완 후 다시 진단하도록 안내

## 입문자 모드 단계

입문자 모드는 결과를 한 번에 모두 펼치지 않고 다음 순서로 보여줍니다.

1. 핵심 문제 유형
2. 먼저 그릴 FBD
3. 사용할 수 있는 핵심 원리
4. 첫 번째로 세울 식
5. 다음 식과 연결 조건
6. 쓰면 안 되는 식
7. 자주 하는 실수
8. 직접 계산해볼 부분

## 식 설명 원칙

각 적용식에는 다음 설명을 붙입니다.

- 힘의 방향
- 좌표축/양의 방향
- 왜 그 항들이 더해지거나 빼지는지
- 어떤 조건에서만 사용할 수 있는지

예: `m_B g - T = m_B a`는 매달린 물체 B를 아래쪽 양의 방향으로 잡았을 때 중력은 +, 장력은 - 방향이므로 뉴턴 제2법칙에서 나옵니다.
