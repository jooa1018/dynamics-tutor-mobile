# 3차원 강체 운동 미지원

현재 버전은 주로 2차원 입자 동역학 및 강체 평면운동 문제를 대상으로 합니다.

## 미지원 유형

- 3D rigid body dynamics
- Euler equations of motion
- gyroscopic motion
- precession / nutation
- inertia tensor matrix problems
- rotating reference frame problems
- Coriolis acceleration in 3D mechanisms

해당 단서가 입력되면 앱은 일반 2D 공식을 적용하지 않고, 현재 미지원 범위임을 안내합니다.
