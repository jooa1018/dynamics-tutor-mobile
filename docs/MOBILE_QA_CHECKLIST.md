# 모바일 QA 체크리스트

- PC Chrome에서 앱 실행
- 모바일 Chrome 또는 Android Chrome 폭 확인
- iPhone Safari 또는 모바일 Safari 폭 확인
- 문제 입력창이 잘리지 않는지 확인
- 결과 카드가 세로로 자연스럽게 쌓이는지 확인
- FBD SVG가 화면 밖으로 넘치지 않는지 확인
- 수식이 줄바꿈/가로스크롤로 읽히는지 확인
- 학생 모드에서 디버그 정보가 숨겨지는지 확인
- API key가 화면/로그/오류에 노출되지 않는지 확인
- APP_PASSWORD 게이트가 정상 동작하는지 확인
- 오답노트 저장/검색/필터/백업이 정상인지 확인
