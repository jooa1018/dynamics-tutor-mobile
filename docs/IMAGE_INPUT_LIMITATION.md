# 이미지/그림 입력 한계

현재 버전은 이미지/그림을 직접 인식하지 않습니다. 교재 그림에 포함된 치수, 각도, 힘 방향, 접촉 조건, 도르래 연결 구조 등을 텍스트로 입력해야 정확한 진단이 가능합니다.

## 그림 정보를 텍스트로 바꾸는 체크리스트

- 물체 개수와 각 물체의 질량
- 경사각 θ, 반지름 R, 끈 길이 L
- 끈/도르래/레일/트랙/접촉면 연결 구조
- 마찰 있음/없음, μ_s 또는 μ_k
- 회전축·피벗·힌지·충돌 위치
- 힘 방향 또는 운동 방향
- 초기속도/각속도와 구하려는 값
- 좌표축 또는 양의 방향 선택

## 예시

나쁜 입력:

> 그림과 같은 시스템에서 가속도를 구하라.

좋은 입력:

> 질량 m_A인 블록 A가 마찰 없는 수평면 위에 있고, 질량 m_B인 블록 B가 이상적 도르래에 매달려 있다. 두 물체의 가속도와 장력을 구하라.

## 향후 확장 가능성

향후 이미지 업로드, OCR, 비전 모델 기반 FBD 추출 기능을 붙일 수 있습니다. 단, 그림 인식은 오판 위험이 있으므로 텍스트 조건 검증 UI와 함께 구현하는 것이 안전합니다.
